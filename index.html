Bars

A bar is enclosed in a element with lcars-bar class and each segment of the bar is defined using lcars-bar--segment class

Segment customization class:

  • expand: fill available space in the bar
  • left-end: create a left rounded edge
  • right-end: create a right rounded edge
  • half-segment: make the segment take half of the height of the bar
  • left-space: add a space on the left of the segment
  • right-space: add a space on the right of the segment
  • double-left-space: add a double space on the left of the segment
  • double-right-space: add a double space on the right of the segment

Example:

<div class="lcars-bar">
    <div class="lcars-bar--segment lcars-melrose-bg left-end"></div>
    <div class="lcars-bar--segment lcars-pale-canary-bg" style="width: 70px"></div>
    <div class="lcars-bar--segment left-space lcars-golden-tanoi-bg" style="width: 50px"></div>
    <div class="lcars-bar--segment left-space half-segment lcars-danub-bg" style="width: 300px"></div>
    <div class="lcars-bar--segment lcars-danub-bg" style="width: 20px"></div>
    <div class="lcars-bar--segment left-space" style="width: 300px"></div>

    <div class="lcars-bar--segment left-space expand"></div>
    <div class="lcars-bar--segment left-space right-end"></div>
</div>

Elbow

Create a lcars typical elbow to link a column and a bar.

Class: lcars-elbow

Elbow orientation class:

  • right-top: elbows faces to the top right.
  • left-top: elbows faces to the top left.
  • right-bottom: elbows faces to the bottom right.
  • left-bottom: elbows faces to the bottom left.
